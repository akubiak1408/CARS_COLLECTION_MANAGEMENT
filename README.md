## General info
About methods.
1. Show you all cars which are saved in JSON file.
2. Sort cars
	Sort type methods
Ascending or Descending
     a) by Model
     b) by Color
     c) by Price
     d) by Mileage


Data generator service generate random data to JSON file.
For example: 
 {
    "model": "JEEP",
    "price": 50208,
    "color": "GREEN",
    "mileage": 68372.06,
    "components": [
      "Cameras",
      "Audio"
    ]
  }
